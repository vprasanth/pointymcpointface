# Slack Marketplace Branding and Logo Checklist

## Logo best practices

- Use a distinctive app icon that does not imitate Slack brand assets.
- Keep the design simple so it remains legible at small sizes.
- Prefer a square PNG as the source asset.
- Keep branding consistent across app icon, screenshots, and public pages.
- Use descriptive alt text where the logo appears on web pages.

## Social preview metadata

For links shared in apps like WhatsApp, Facebook, iMessage, X/Twitter, and many others, use:

- `meta name="description"`
- `meta property="og:type"` (`website`)
- `meta property="og:site_name"`
- `meta property="og:title"`
- `meta property="og:description"`
- `meta property="og:image"`
- `meta property="og:image:type"`
- `meta property="og:image:width"`
- `meta property="og:image:height"`
- `meta property="og:image:alt"`
- `meta name="twitter:card"` (`summary`)
- `meta name="twitter:title"`
- `meta name="twitter:description"`
- `meta name="twitter:image"`
- `meta name="twitter:image:alt"`
